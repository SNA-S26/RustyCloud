## RustyCloud

### Usage
- To run the application, install [Rust](https://doc.rust-lang.org/book/ch01-01-installation.html).
- Run the application with `cargo`:
    ```bash
    cargo run -- <port>
    ```

    Alternatively, run the application with the command _(port `80` will be chosen by default)_:

    ```bash
    cargo run
    ```
